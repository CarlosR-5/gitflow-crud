# CRUD GitFlow Project

Este proyecto consiste en una aplicación web básica para la gestión de usuarios (CRUD), desarrollada con HTML, CSS y JavaScript.

## Funcionalidades
- Inicio de sesión (login)
- Crear usuarios
- Editar usuarios
- Eliminar usuarios
- Validación de formularios

## Metodología de trabajo
Se utilizo la metodología **Git Flow**, utilizando las siguientes ramas:

- `main`: versión final del proyecto
- `develop`: integración de desarrollo
- `qa`: pruebas
- `feature/*`: nuevas funcionalidades
- `hotfix/*`: corrección de errores

Cada funcionalidad fue desarrollada en su propia rama y luego integrada mediante Pull Requests hacia `develop`, `qa` y `main`.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- Git & GitHub
